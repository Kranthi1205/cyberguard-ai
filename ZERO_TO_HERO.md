# 🛡️ CyberGuard AI: The Ultimate Zero to Hero Guide

Welcome to the definitive guide for **CyberGuard AI**, a state-of-the-art, agentic Security Operations Center (SOC) platform. This document will transform you from a beginner to an expert in the system's architecture, logic, and deployment.

---

## 🚀 1. Executive Summary
CyberGuard AI is an autonomous security platform that combines the **MERN stack** with **Groq's LPU™ technology**. It doesn't just display logs; it acts as a **24/7 AI Security Analyst**, automatically analyzing threats, recommending response playbooks, and managing security incidents with enterprise-grade precision.

---

## 🛠 2. Technical Architecture (The Engine Room)

### The Backend (The Brain) - `Node.js & Express`
The backend follows a strict **Controller-Route-Model** architecture:
*   **Middleware (`/middleware`)**: Guards the gates. The `protect` middleware verifies JWT tokens for every request, ensuring only authenticated users can access security data.
*   **Controllers (`/controllers`)**: The logic center. 
    *   `aiController.js`: Manages the bridge to the Groq API.
    *   `threatController.js`: Handles threat lifecycle, stats, and automated responses.
    *   `authController.js`: Manages Google OAuth and secure local login.
*   **Models (`/models`)**: The data blueprints using Mongoose (MongoDB).
    *   `User.js`: Stores user profiles, roles (Admin/Analyst), and hashed passwords.
    *   `Threat.js`: Detailed records of every security incident.
    *   `SiemLog.js`: Raw event data from simulated firewalls/endpoints.

### The Frontend (The Face) - `React.js`
A modern, responsive dashboard built with **Vanilla CSS** and **Recharts**.
*   **Context API**: Uses `AuthContext.js` to manage global user state and persistence.
*   **Responsive Design**: A fluid grid system that adjusts seamlessly between 4K monitors and mobile browsers.
*   **Glassmorphism UI**: Uses background blurs and sleek dark-mode tokens for a premium enterprise feel.

## 🤖 3. AI Intelligence: How It "Thinks"
We use **Groq's LLaMA 3.3-70B** model. Here is the magic:
1.  **The Agent Identity**: Every AI call is wrapped in a "System Prompt" that forces the AI to behave as a technical SOC analyst.
2.  **Context Injection**: When analyzing a threat, we don't just ask "what is this?". We inject the **Source IP, Protocol, Severity, and Category** into the prompt.
3.  **Low Latency**: Thanks to Groq's LPUs, the AI generates complex security playbooks in **under 2 seconds**.

## 🔒 4. Security & Authentication
*   **Bcrypt.js**: Passwords are never stored as plain text. They are salted 12 times and hashed.
*   **JWT (JSON Web Tokens)**: Once logged in, you receive a signed token valid for 7 days.
*   **Google Identity Services**: Uses Google's modern GSI library for secure, one-tap login without password fatigue.
*   **CORS Protection**: The backend is hardcoded to only allow requests from your specific Vercel production URL.

## 📊 5. Data Modeling
### The Threat Object
Every threat is a rich object containing:
*   `title`, `severity`, `category`, `status`
*   `sourceIP` / `destinationIP`
*   `aiAnalysis`: A stored long-form report generated by the AI.
*   `automatedResponseApplied`: A boolean flag for SOC automation.

## 🌐 6. Deployment & Dev-Ops
The app is built to be "Headless" (Frontend and Backend are separate).

### Backend (Render)
*   **Root Directory**: `backend`
*   **Build Command**: `npm install`
*   **Start Command**: `npm start`
*   **Crucial Var**: `MONGODB_URI` & `GROQ_API_KEY`

### Frontend (Vercel)
*   **Root Directory**: `frontend`
*   **Build Command**: `npm run build`
*   **Output Directory**: `build`
*   **Crucial Var**: `REACT_APP_API_URL` (Point this to your Render URL)

## 🆘 7. Troubleshooting & FAQs

### "CORS Error" (The #1 Issue)
**Problem:** Frontend cannot talk to Backend.
**Fix:** Ensure your Render backend has `CLIENT_URL` set to your Vercel address. I have also hardcoded the fix into the latest `index.js` to make it foolproof!

### "Google Sign-In Failed"
**Problem:** The button doesn't log you in.
**Fix:** You MUST add your Vercel URL to the **Authorized JavaScript Origins** in your Google Cloud Console.

### "AI says MOCK MODE"
**Problem:** The AI responses are generic placeholders.
**Fix:** You haven't added a `GROQ_API_KEY` to your Render environment variables.
